作品名：ISLAND
制作時間：二か月ほど
作品タイプ：個人作品
環境：Windows、DirectX９

＜操作方法＞：
カメラ：	     マウスの右ボタン押しながら：カメラ回転
	     マウスの右ボタン押しながらホイール回転：カメラをズームさせる

フィールドエディター：WSADで調整したい地面の選択範囲を移動させる
	      ↑、↓　で地面の高さを調整する

モデルエディター：  WSADでオブジェクトの位置を調整する


＜アピールポイント＞：
初めてC++でエディターを作りました。

デザインパータンは
http://gameprogrammingpatterns.com/component.html
を参考しました。

制作中、四年生の先輩と先生からいろいろ教えてもらいました。
